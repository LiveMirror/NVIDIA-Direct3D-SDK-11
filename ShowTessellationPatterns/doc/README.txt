

Overview:
==================
This sample demonstrates the tessellation patterns on triangle and quad domain.


UI can be used to:
==================
- Select Tessellation Mode:
  * Integer
  * power2
  * Fractional Odd
  * Fractional Even

- Specify Tessellation Factor per edge
  * 3 edges per triangle, 4 edges per quad
  * up to 64

- Specify Interior Tessellation factor
  * Max
  * Min
  * Average
